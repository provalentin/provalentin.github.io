# Interview Preparation: Resume Tips for Tech Interviews

**Read time:** 8 min | **Reference:** Yes

## Format & Structure

```
One page (early career) to 2 pages (10+ years)

Order:
├─ Contact info (name, email, phone, city, LinkedIn)
├─ Professional summary (optional, 1-2 lines)
├─ Experience (most recent first)
├─ Education
├─ Technical skills
└─ Projects (optional)
```

## Experience Section

### Strong Bullet Points

```
❌ Weak: "Worked on backend systems"
✓ Strong: "Designed and implemented caching layer,
           reducing API latency from 500ms to 100ms (5x improvement)
           and increasing throughput to 50K RPS"

Structure: [Action] + [What you did] + [Quantified result]
```

### Action Verbs

```
Designed, implemented, architected, optimized,
led, managed, mentored, improved, reduced,
increased, decreased, developed, created,
coordinated, enhanced, accelerated, pioneered
```

### Quantify Everything

```
Don't say: "Improved performance"
Say: "Reduced database query time by 60% (2sec → 0.8sec)"

Don't say: "Worked on large project"
Say: "Led team of 5 engineers building real-time analytics"

Numbers = Impact
```

### For Each Job

```
Company | Title | Dates

3-5 bullet points:
1. Describe the project/system you worked on
2. Your specific contribution (you, not "we")
3. Quantified impact (faster, cheaper, more users, etc)
4. Technical skills demonstrated
5. Optional: challenge overcome or learning

Example:
"Built distributed cache layer using Redis
for product recommendation engine,
reducing database load by 70%
and improving recommendation latency from 5s to 200ms.
Served 50M+ queries daily across 3 data centers."
```

## Education

```
University Name
B.S. in Computer Science | Graduation: May 2020

Optional:
├─ GPA (only if 3.8+)
├─ Relevant coursework
├─ Honors/scholarships
└─ Academic projects

Keep it short if recent graduate
Remove after 5 years of experience
```

## Technical Skills

```
Languages: Python, Java, C++, JavaScript, Go
Frameworks: Django, Spring, React, FastAPI
Databases: PostgreSQL, MongoDB, Redis, Cassandra
Tools: Docker, Kubernetes, AWS, GitHub
Concepts: Distributed systems, Machine learning, APIs

Don't list technologies you barely know
Prioritize what's relevant to target role
```

## What to Include

```
✓ Quantified impact ("5x faster", "$2M saved")
✓ Technical leadership ("led team of 5")
✓ Complexity ("designed for 1M concurrent users")
✓ Industry context ("e-commerce platform", "ML platform")
✓ Results (grew users, reduced costs, improved quality)
```

## What to Exclude

```
❌ Responsibilities (job description already covers)
❌ Weak action verbs ("was involved", "helped")
❌ Acronyms without context
❌ Too much technical jargon
❌ Vague achievements
❌ Personal pronouns ("I was...")
❌ Dates on skills
```

## Common Mistakes

### ❌ Long Sentences

```
❌ "Designed, implemented and optimized a highly scalable
    microservices architecture that leverages containerization
    technologies and cloud-native infrastructure patterns"

✓ "Built microservices architecture using Docker & Kubernetes,
   scaling to handle 100K concurrent users"
```

### ❌ Weak Verbs

```
❌ "Responsible for"
❌ "Worked on"
❌ "Helped with"
❌ "Involved with"

✓ "Led", "Designed", "Architected", "Optimized"
```

### ❌ No Numbers

```
❌ "Improved performance of recommendation algorithm"

✓ "Improved recommendation algorithm efficiency by 40%
   (reduced training time from 2 days to 20 hours)"
```

## For Different Experience Levels

### Entry Level (0-2 years)

```
Focus on:
├─ Projects (hackathons, side projects, academic)
├─ Technical skills
├─ What you learned
└─ Internships heavily weighted

Example project:
"Built real-time chat application using WebSocket and React,
deployed to AWS, handling 1K concurrent connections"
```

### Mid-Level (2-5 years)

```
Focus on:
├─ Impact and results (with numbers)
├─ Technical depth
├─ Ownership of projects
└─ Team contributions

Example:
"Owned recommendation engine, improved relevance by 25%,
increased click-through rate 40%, serving 100M queries/day"
```

### Senior (5+ years)

```
Focus on:
├─ Strategic impact
├─ Leadership and mentoring
├─ Organization influence
└─ Scope of projects

Example:
"Led 10-person engineering team building data platform,
drove $5M cost savings through infrastructure optimization,
mentored 3 engineers who got promoted"
```

## Keywords for ATS

```
Include keywords from job description:
├─ Technologies mentioned ("Kubernetes", "React")
├─ Soft skills ("Communication", "Leadership")
├─ Domain ("Machine learning", "Real-time")
└─ Metrics ("Scalability", "Performance")

Many resumes filtered by keyword matching
Match job description naturally
```

## Final Checklist

```
✓ One consistent format/font
✓ No typos or grammatical errors
✓ Quantified achievements with numbers
✓ Strong action verbs
✓ Technical depth appropriate to level
✓ Fits on 1 page (junior) or 2 pages (senior)
✓ Sorted by relevance to target role
✓ No obvious gaps explained
✓ Contact info at top
✓ PDF format (preserves formatting)
```
